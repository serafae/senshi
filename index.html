PURPOSE
This is a fan-made archive dedicated to Sailor Moon character profiles. It serves as a personal collection of information, references, and character pages organized for easy viewing and future expansion.

INFO
Each page in this collection represents a different Sailor Senshi, featuring basic profile information and space for future images, notes, and detailed character content. The site is currently in early development and will be expanded over time.

This site is currently under development and is being built piece by piece. The goal is to create a structured archive of Sailor Moon characters that can be expanded over time with detailed profiles, visual elements, and personalized formatting. The current version prioritizes organization and accessibility over visual design.

Layout made as a temporary structure for GitHub Pages hosting. Content and design will be updated and improved over time.

NAVIGATION
Each section represents a different part of the Sailor Moon universe, allowing characters to be explored based on their role, relationships, and affiliations within the story.
