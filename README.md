<center><h1><i><b>Serafae's Senshi</b></i></h1></center>

This project is only recreational and serves as a fun, exciting coding practice.

The purpose of this project was <i>never</i> meant for role-playing purposes; rather, it was meant to showcase intentional, stylistic design & layout manipluation on SpaceHey.

This GitHub Pages project is a <i>Sailor Moon</i> fan-based project meant to reflect each character closely to canon of the main Senshi.

No side characters will be included due to regional constraints.
Because platform access and regulations can change, I'm keeping the project focused and archiving it on GitHub, so the original work won't be lost.

<b><i>Serafae's Senshi</i></b> will be imported and exported to GitHub to <b>100%</b> by completion.
